## 🚀 Project Highlights

### 1️⃣ 🏠 House Price Prediction
> *Predict property values with state-of-the-art regression models*

**Features:**
- 📊 Multiple algorithms: Linear Regression, Random Forest, XGBoost
- 💰 Real-time predictions via sleek Flask web app
- 🎨 Modern, responsive UI with dark/light mode
- 📈 Feature importance visualization

**Quick Start:**
```bash
cd house-price-prediction
pip install -r requirements.txt
python app.py
```

---

### 2️⃣ 🔍 Plagiarism Detector *(LIVE WEB APP)*
> *"Copy-paste once, get caught forever."* ⚡

**Features:**
- 🎯 **95%+ accuracy** using TF-IDF + Logistic Regression
- 🌟 **Cyberpunk glassmorphism design** with neon accents
- 🌙 Dark mode with smooth animations
- ⚡ Real-time detection & similarity scoring
- 🛡️ Trained on real plagiarism dataset

**Quick Start:**
```bash
cd plagiarism-detector
pip install flask scikit-learn nltk pandas
python app.py
# Visit: http://127.0.0.1:5000
```

*"Turn in someone else's work? The model knows."* 🔥

### 3️⃣ 🌸 Iris Flower Classification
> *The legendary ML project — supercharged* ✨

**Features:**
- 🧠 Multiple algorithms: Logistic Regression, KNN, SVM, Decision Tree
- 📈 Beautiful visualizations (confusion matrix, pair plots)
- 🎯 99.9% accuracy on test set
- 📊 Interactive classification reports

**Quick Start:**
```bash
cd iris-flower-classification

```
