# Machine Learning Projects Collection (3-in-1)

Three fully working, production-ready ML projects in one repo:

1. House Price Prediction (Regression)  
2. Plagiarism Detector (NLP + Web App)  
3. Iris Flower Classification (Classic ML)

All projects include beautiful web interfaces (where applicable), clean code, and are ready to run in seconds.

### Project 1: House Price Prediction
Predicts house prices using features like area, bedrooms, location, etc.

- Models: Linear Regression, Random Forest, XGBoost (you can switch)
- Real-time prediction via Flask web app
- Modern, responsive UI

**Run:**
```bash
cd house-price-prediction
pip install -r requirements.txt
python app.py
